Problemas para abrir os slides das aulas? Aqui estão algumas dicas para resolver isso:

Baixe os arquivos pelo computador, não pelo celular. Ao baixar pelo celular, os arquivos podem ficar desconfigurados.

Verifique se você tem um leitor de PDF instalado no computador. Caso contrário, recomendamos instalar o Adobe Acrobat Reader DC (disponível gratuitamente). Esse programa irá abrir automaticamente todos os arquivos PDF que você baixar.
